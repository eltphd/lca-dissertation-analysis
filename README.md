# Latent Class Analysis: Black Adolescent Discrimination Experiences

Code for dissertation: "Unraveling Hopelessness: A Latent Class Analysis of Black Adolescent Student Experiences"

## Methods
- **Latent Class Analysis (LCA)** using poLCA package
- **Model comparison** via BIC, entropy, classification diagnostics
- **CDC YRBSS data** (n=7,000+)

## Key Findings
Identified 4 distinct profiles of discrimination exposure with differential mental health outcomes.

## Files
- `01_data_prep.R` - Data cleaning and recoding
- `02_lca_models.R` - Model estimation and comparison
- `03_visualizations.R` - Class profile plots
- `04_validation.R` - Cross-validation and sensitivity analyses
